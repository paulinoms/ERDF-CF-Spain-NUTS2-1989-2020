# ERDF-CF-Spain-NUTS2-1989-2020
This dataset compiles certified expenditure data from ERDF and Cohesion Fund in Spain, covering six  programming periods from 1989 to 2020. Data are disaggregated by NUTS2 regions, year, programming axis,  and thematic area. All monetary values are expressed in thousands of  euros at constant 2000 and 2020 prices , using the INE as the deflator.
